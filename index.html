The semantic tag refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages. The following HTML tags can be used to break your page into identified parts:-
header: It defines a header for a web page. nav: It defines a container for navigation links. section: This defines a section in a web page. footer: It defines a footer for a document or a section.

If two sets of HTML tags are overlapped, only the first tag will be recognized. You will find this problem when the text does not display properly on the browser screen.
